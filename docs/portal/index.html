RMath User Guide

RMath is an industrial-grade numerical toolkit for Python, built from the ground up in Rust. It provides high-performance data structures and mathematical kernels that scale across multi-core CPUs while maintaining strict IEEE-754 precision.

The RMath Difference

Unlike traditional C-based libraries, RMath utilizes Rust's memory safety and modern concurrency primitives (Rayon) to bypass the Python GIL without the risks of manual thread management.

basic_usage.py 
import rmath as rm

# All operations scale automatically
data = rm.Array.randn(1000000)
result = data.sin().mean()

print(f"Result: {result}")
