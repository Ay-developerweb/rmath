rmath v0.1.0 · Rust + PyO3
v0.1.0 · Tier-1 Documentation

The Fastest Numerical Engine for Modern Python.

RMath is a high-performance numerical toolkit backed by Rust. Leveraging Rayon for zero-GIL parallelization, it achieves 100% benchmark parity with NumPy while providing a modern, memory-safe analytical engine.

Explore Modules → Installation Guide
demo.py 
import rmath as rm

# Initialize high-performance parallel array
data = rm.Array.linspace(0, 100, 1_000_000)

# Zero-GIL parallel processing across all cores
result = data.sin().exp().sum()

print(f"Result: {result}")

Explore the Toolkit

Every module is optimized for parallel execution and Tier-1 documented.

Vector Engine

SIMD-accelerated 1D math. Support for complex numbers, trigonometry, and statistical reductions.

Array Engine

N-dimensional array with triple-tier storage: Inline (Stack), Heap (Shared), and Mmap (Lazy).

Statistical Suite

High-precision descriptive and inferential statistics using Welford's stable algorithm.

Numerical Calculus

Automatic differentiation via Dual numbers, high-order integration, and root solvers.

Spatial Geometry

3D Transformations, Quaternions, and topological algorithms like Convex Hull.

Signal Processing

O(N log N) FFT-based convolutions and multi-mode Fourier analysis.

Linear Algebra

Matrix solvers, SVD, QR, and Cholesky decompositions powered by faer.

Machine Learning

SIMD activation functions, loss gradients, and normalization layers.

Special Functions

Gamma, Error, and Beta functions with full broadcasting support.

Scalar Core

Precision-optimized 64-bit math for single values and fundamental calculations.
