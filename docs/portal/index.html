v0.1.4 — Beta

Drop-in accelerator
for Python computing.

RMath offloads heavy mathematical workloads to Rust and integrates seamlessly back into Python via PyO3. Array operations, autograd, linear algebra, and statistics all execute outside the GIL on a Rayon thread pool.

Explore Modules → Install GitHub
demo.py 
import rmath as rm

# Create a 1000×1000 random matrix — runs in Rust
data = rm.Array.randn(1000, 1000)

# Parallel stats — executes outside the GIL
avg = data.mean()
std = data.std_dev()

# Autograd — forward + backward in Rust
x = rm.Tensor.randn(100, 100, requires_grad=True)
y = (x.sigmoid() * x).sum()
y.backward()
print(x.grad)  # gradients computed in pure Rust

# Zero-copy bridge to NumPy
np_arr = data.to_numpy()

Installation

Pre-built wheels are available for Windows, Linux, and macOS. No Rust toolchain required.

$ pip install rmath-py
Note: The package is published as rmath-py on PyPI but you import it as import rmath.

Quick Start

N-Dimensional Arrays

arrays.py 
import rmath as rm

a = rm.Array([[1, 2, 3],
              [4, 5, 6]])
b = rm.Array.randn(2, 3)

c = a + b              # element-wise add
d = a.sin().exp()      # chained element-wise ops
avg = a.mean()         # global reduction → 3.5

# Works with any dimension
vol = rm.Array.zeros(16, 3, 256, 256)  # 4-D tensor

Autograd (Tensor)

autograd.py 
import rmath as rm

# Training-ready tensors with gradient tracking
W = rm.Tensor.randn(3, 3, requires_grad=True)
x = rm.Tensor.randn(3, 1)

y = (W @ x).sigmoid().sum()
y.backward()

print(W.grad)  # ∂y/∂W — computed via reverse-mode AD

Ecosystem Interop

interop.py 
import rmath as rm
from sklearn.linear_model import LinearRegression

# Generate data in Rust, feed directly to scikit-learn
X = rm.Array.randn(100, 1)
Y = rm.Array.randn(100, 1)

model = LinearRegression()
model.fit(X, Y)  # Just Works™ — __array__ protocol

# Explicit bridges
np_arr = X.to_numpy()
torch_t = X.to_torch()

stats_example.py 
import rmath as rm

data = [2.4, 3.1, 2.8, 3.5, 2.9, 3.2]

Architecture

architecture 
Python (rmath)  ──PyO3──▸  Rust core
                              │
                  ┌───────────┼───────────┐
                  ▼           ▼           ▼
               Stack       Heap       Mmap
             (inline)    (shared)    (lazy I/O)
                  │           │           │
                  └─── Rayon thread pool ──┘
No GIL: Heavy loops release the Python GIL and execute across all CPU cores via Rayon. Small arrays stay on the stack for zero-allocation speed.
Flat Storage: Arrays store data in a contiguous Vec<f64> buffer with a separate shape vector. This enables cache-friendly access patterns and efficient N-dimensional reshaping without data movement.
