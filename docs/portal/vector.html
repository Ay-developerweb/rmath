Core Module

rmath.vector

RMath's flagship 1D engine. Features over 150 parallelized operations categorized into arithmetic, trigonometry, hyperbolic, and logical filters.

Rayon Parallel

Heavy loops are automatically offloaded to the Rayon thread pool with zero GIL overhead.

Operator Overloading

Native Python support for +, -, *, /, @, and unary negation for both Scalar and Vector operands.

Constructors

import rmath as rm

v = rm.Vector.range(0, 10, 0.1)
lin = rm.Vector.linspace(0, 1, 100)
rand = rm.vector.randn(1000)

API Reference

Method Complexity Description
v.sum() O(N) Parallel Kahan-sum reduction.
v.sin() O(N) Element-wise parallel sine.
v.unique() O(N log N) Return sorted unique elements.
v.argsort() O(N log N) Parallel unstable sort indexer.
