Analytical Module

rmath.stats

High-precision statistical analysis using parallelized Welford's algorithm and O(N) selection for medians. Designed for massive datasets.

Numerical Stability

RMath uses Welford's Online Algorithm for calculating variance and kurtosis. Unlike the naive sum-of-squares approach, Welford's method is incredibly stable even for extremely large numbers or datasets with high variance, and it works in a single parallel pass.

Descriptive Statistics

import rmath.stats as st

data = [1.5, 2.3, 4.7, 9.1, 6.2]

# Fast parallel report
report = st.describe(data)

print(f"Mean: {report['mean']}")
print(f"Kurtosis: {st.kurtosis(data)}")
