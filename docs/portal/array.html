Core Module

rmath.array

The foundation of RMath's N-Dimensional computing. Supports SIMD-accelerated math kernels, automatic broadcasting, and zero-copy interop with Python's scientific ecosystem.

Triple-Tier Storage

Inline

Stack-allocated for < 32 elements. Zero heap allocation. Performance matches raw CPU registers.

Heap

Shared Arc memory for massive datasets. Reference-counted and thread-safe for parallel processing.

Mmap

Disk-backed memory mapping. Process GB-scale datasets without exceeding RAM limits.

Constructors

# Zero-initialized
arr = rm.Array.zeros(100, 100)

# Identity matrix
eye = rm.Array.eye(10)

# From Python data
data = rm.Array([[1, 2], [3, 4]])
