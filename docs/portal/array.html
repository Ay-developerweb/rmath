rmath.array API Reference

The rmath.array module is the foundation of RMath. It provides parallel N-dimensional arrays with shared memory tiers.

Creation

creation.py 
import rmath.array as ra

# Multiple constructors
z = ra.zeros(5, 5)
o = ra.ones(10, 10)
r = ra.randn(100, 100)

print(f"Random mean: {r.mean():.4f}")
Random mean: -0.0012

API Reference

FunctionDescription
zeros(*shape)Array of zeros.
ones(*shape)Array of ones.
randn(*shape)Normal random samples.
arange(s, e, d)Step-wise sequence.
