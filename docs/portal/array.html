Core Module

rmath.array

N-dimensional array engine. Row-major Vec<Vec<f64>> backed, with all element-wise operations parallelized via Rayon. Supports automatic storage tiering: inline (stack), heap, and memory-mapped files.

example.py
import rmath as rm

a = rm.Array([[1, 2], [3, 4]])
b = rm.Array.eye(2)

c = a @ b          # matrix multiply
d = a.sin().exp()  # element-wise chain
avg = a.mean()     # global reduction
