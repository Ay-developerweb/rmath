Linear Algebra

rmath.linalg

Industrial-grade matrix solvers and decompositions powered by the faer and matrixmultiply libraries.

Core Matrix Operations

Function Description
inv(A) Matrix inverse via LU Decomposition with partial pivoting.
det(A) Compute matrix determinant.
solve(A, B) Solve Ax = B for x.
svd(A) Singular Value Decomposition (Returns U, S, V_T).
qr(A) QR Decomposition (Returns Q, R).
cholesky(A) Cholesky Decomposition for positive-definite matrices.
pseudo_inv(A) Moore-Penrose pseudo-inverse using SVD.
