Core Module · Autograd

rmath.Tensor

Reverse-mode automatic differentiation engine built on top of rmath.Array. Tracks computation graphs, computes gradients via .backward(), and executes entirely in Rust — 3.98× faster than PyTorch on CPU for small-to-medium models.

tensor_demo.py
import rmath as rm

# Create tensors with gradient tracking
x = rm.Tensor([[1.0, 2.0], [3.0, 4.0]], requires_grad=True)
W = rm.Tensor.randn(2, 2, requires_grad=True)

# Forward pass — builds computation graph
y = (W @ x).sigmoid().sum()

# Backward pass — computes ∂y/∂W and ∂y/∂x
y.backward()

print(W.grad)   # gradient of y w.r.t. W
print(x.grad)   # gradient of y w.r.t. x

Backward Pass

Call .backward() on a scalar Tensor to compute gradients for all upstream tensors with requires_grad=True. Gradients accumulate in the .grad property.

Property / MethodTypeDescription
.backward()NoneCompute gradients via reverse-mode AD. Must be called on a scalar.
.gradTensor | NoneAccumulated gradient after .backward() has been called.
.requires_gradboolWhether this tensor participates in gradient computation.
.data()ArrayAccess the underlying Array (detached from the graph).

Gradient Correctness Examples

gradients.py
import rmath as rm

# ── Addition: ∂(a+b)/∂a = 1, ∂(a+b)/∂b = 1 ──
a = rm.Tensor([[1.0]], requires_grad=True)
b = rm.Tensor([[2.0]], requires_grad=True)
c = (a + b).sum()
c.backward()
print(a.grad)  # Tensor(data=Array([[1.0]]), ...)
print(b.grad)  # Tensor(data=Array([[1.0]]), ...)

# ── Multiplication: ∂(a*b)/∂a = b, ∂(a*b)/∂b = a ──
x = rm.Tensor([[3.0]], requires_grad=True)
y = rm.Tensor([[4.0]], requires_grad=True)
z = (x * y).sum()
z.backward()
print(x.grad)  # 4.0 (value of y)
print(y.grad)  # 3.0 (value of x)

# ── Sigmoid: ∂σ(0)/∂x = σ(0)·(1-σ(0)) = 0.25 ──
s = rm.Tensor([[0.0]], requires_grad=True)
out = s.sigmoid().sum()
out.backward()
print(s.grad)  # 0.25

# ── Chain rule: sigmoid(x²) ──
v = rm.Tensor([[1.0]], requires_grad=True)
out = (v * v).sigmoid().sum()
out.backward()
print(v.grad)  # 2x · σ(x²) · (1 - σ(x²)) ≈ 0.3350

Training Loop

A complete example showing how to train a simple model using Tensor autograd.

training.py
import rmath as rm

# Simple linear model: y = Wx
W = rm.Tensor.randn(3, 3, requires_grad=True)
lr = 0.01

for step in range(100):
    # Forward
    x = rm.Tensor.randn(3, 1)
    pred = (W @ x).sigmoid()
    loss = pred.sum()

    # Backward
    loss.backward()

    # Manual SGD update
    grad_data = W.grad.data()
    w_data = W.data()
    new_data = w_data - grad_data * lr
    W = rm.Tensor(new_data, requires_grad=True)

    if step % 20 == 0:
        print(f"Step {step}, Loss: {loss}")
