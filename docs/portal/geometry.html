Spatial Module

rmath.geometry

Distance metrics, quaternion-based rotations, and topological algorithms for computational geometry and robotic kinematics.

Vector Relationships

Function Algorithm
euclidean(v1, v2) Standard L2 norm distance.
cosine_similarity(v1, v2) Angular similarity between vectors.
cross_product(v1, v2) Computing the normal vector in 3D space.
angle_between(v1, v2) Compute radians between two vectors.

3D Rotations (Quaternions)

from rmath.geometry import Quaternion

# Create a rotation of 90 degrees around Z axis
q = Quaternion.from_axis_angle([0, 0, 1], 1.5708)

# Apply rotation to a vector
v = [1, 0, 0]
v_rotated = q.rotate_vector(v) # -> [0, 1, 0] approx

Topological Operations

Function Algorithm
is_point_in_polygon(p, poly) Efficient Ray Casting implementation.
convex_hull(points) Monotone Chain O(N log N) implementation.
