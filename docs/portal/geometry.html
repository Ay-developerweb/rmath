rmath.geometry

Computational geometry primitives and spatial transformations.

Proven: Quaternion Rotation

geometry_verify.py 
import rmath.geometry as geo
import rmath.constants as c

# Rotate 90 degrees around Z axis
q = geo.Quaternion.from_axis_angle([0, 0, 1], c.pi / 2)
p = geo.Point3(1.0, 0.0, 0.0)

rotated = q.rotate(p)
print(f"Rotated Point: {rotated}")
Rotated Point: Point3(0.0000, 1.0000, 0.0000)
