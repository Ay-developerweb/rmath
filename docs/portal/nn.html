Machine Learning

rmath.nn

High-performance Neural Network primitives. Features SIMD-parallel activations, losses, and normalization layers for deep learning backends.

Activations

Function Description
relu(A) Rectified Linear Unit element-wise.
sigmoid(A) Standard logistic sigmoid function.
gelu(A) Gaussian Error Linear Unit (GPT-style approximation).
softmax(A) Numerically stable row-wise softmax.

Loss Functions

from rmath import nn

preds = rm.Array.randn(10, 5)
targets = rm.Array.zeros(10, 5)

# Mean Squared Error
loss = nn.mse_loss(preds, targets)
