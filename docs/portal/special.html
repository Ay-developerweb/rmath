Mathematical Module

rmath.special

Special mathematical functions (Gamma, Error, etc.) with support for broadcasting across Scalars, Vectors, and Arrays.

Special Functions

Function Complexity Description
gamma(x) O(1/N) Euler Gamma function Γ(x).
ln_gamma(x) O(1/N) Natural logarithm of Gamma, ln|Γ(x)|.
erf(x) O(1/N) Error function for probability distributions.

Flexible Dispatching

from rmath import special, Vector, Array

# Scalar call
special.gamma(5.0) # 24.0

# Vector call (Parallel)
v = Vector.range(1, 10)
special.gamma(v)

# Array call (Parallel)
a = Array.ones(20, 20)
special.gamma(a)
